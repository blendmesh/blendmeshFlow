# tab component

Example of use.

```js
    <Tabs tabbed={["tab1","tab2", "tab3"]}>
        <TabContent content="tab1">
            <p>This is the first content.</p>
        </TabContent>

        <TabContent content="tab2">
            <p>This is the first content.</p>
        </TabContent>

        <TabContent content="tab3">
            <p>This is the first content.</p>
        </TabContent>
    </Tabs>
```
